### Explicações sobre o nosso Desafio
- Nosso grupo ficou responsavel por representar uma Fintech voltada para empréstimo para pessoa física; 
- A partir dela criamos a PoupeCash, uma empresa com foco em repassar conteúdo sobre fianças , empréstimos, investimentos e etc.
- Enquanto aguardávamos o layout dos Hipsters, adiantamos a compra do domínio logo que definimos o nome da empresa;
- Começamos a dividir as tarefas baseados na melhor forma de juntar e compartilhar os conhecimentos dos membros da equipe.

![](https://media3.giphy.com/media/Vzk5PFo9iH5AEustmv/200w.webp?cid=ecf05e4796d70c4b5c5ab9189839c893689fec969ad37f22&rid=200w.webp)

### Tecnologias que usamos
- HTML
- CSS
- Bootstrap
- JavaScript

### Como nos organizamos 
- Conseguimos trabalhar nesse único repositório e como tinhamos diferentes niveis de conhecimento em cada área, usamos apenas a branch master da seguinte forma:
    - Dois integrantes ficaram com a parte de HTML e CSS, trabalhando de forma simultânea (pair programing) um auxiliando o outro nas maiores dificuldades;
    - O Terceiro integrante ficou responsável pela parte de JS (contador e PopUp de saida);
- Dessa forma conseguimos trabalhar de forma ágil e conseguimos deixar a página pronta ainda na segunda feira (05/07);
- Apos a conclusão de todos os pontos do checklist, nos dedicamos ao codereview tentando melhorar os detalhes do codigo.

### Highlights
- Tentamos ao máximo utilizar uma boa semântica no código, tanto para SEO quanto para acessibilidade.
- Para a utilização de leitores de telas nós deixamos o código na ordem que ele tem que ser lido
- Utilizamos label nos inputs para deixar claro aos usuários com deficiência o que deve ser preenchido em cada campo
